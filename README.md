OIBSIP_Java_Task4 - 
Online Examination System

Objective : 
Develop a comprehensive online examination platform with user authentication, timed MCQ tests, and automated evaluation with grading system.

 Features :
- Secure login and profile management
- 10 Multiple Choice Questions
- 5-minute countdown timer
- Question navigation (Next/Previous)
- Auto-submit on timeout
- Answer persistence across questions
- Instant result calculation
- Grade assignment (A+ to F)
- Pass/Fail status determination

Tools & Technologies Used : 
- Language:Java
- GUI Framework: Java Swing
- Text Editor : Visual Studio Code
- Components: CardLayout, JRadioButton, ButtonGroup, Timer
- Data Structures: ArrayList, Arrays

Steps Performed : 
1. Designed multi-screen flow using CardLayout
2. Created Question class for MCQ data structure
3. Implemented secure login system
4. Developed profile update functionality
5. Built question bank with 10 MCQs
6. Implemented Timer with countdown mechanism
7. Created question navigation system
8. Developed answer tracking using arrays
9. Built evaluation algorithm
10. Implemented grading system (A+ to F)
11. Created result display screen

How to Run : 
1. Clone the repository
```bash
   git clone https://github.com/Divya2006-developer/OIBSIP_Java_Task4.git
```
2. Open `OnlineExaminationSystem.java` in your IDE
3. Compile and run the program
4. Login with any username/password
5. Update profile (optional)
6. Start the examination
